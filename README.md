# trinigy_vision_engine_v3d_converter
This tool can convert Trinigy Vision Engine Scnery files from .v3d to .obj files.

## Dependencies
You need to have Python installed on your system.

## How to use
1) Put the .v3d file in the same folder as the Python script.
2) Open the Python Script with a text editor and rename the .v3d file variable in line 3 to the name of your .v3d file you want to convert.
3) Open a command line tool like CMD and navigate to the folder that contains the Python Script and the .v3d file you want to convert. Use the command "python v3d_scenario_converter.py" to start the process.
4) After finished the conversion you can find the .obj and .mtl file in the folder. You can than open the files with a 3d software like Blender.
